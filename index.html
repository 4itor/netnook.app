Help

Netnook.app is a lightweight, privacy-first speed dial homepage.

It is a fully browser-based, open source static site.

Companion Chrome extension is available to set Netnook.app as your new tab page.

Features

  • Fast keyboard-first search and filter.
  • As you type, links are filtered in real time to those containing the entered characters.
  • URL detection and direct open mode from the search prompt.
  • Shows the result of any typed calculations, use Enter to copy.
  • Edit mode to reorder, add, edit, and delete links and separators.
  • Import and export bookmarks as JSON backups.

Privacy and behavior

  • All data is stored locally in your browser (localStorage only).
  • User must use the Download option to export a backup JSON file (from the edit mode menu).
  • Backups can be loaded manually to sync with other desktops/browsers.

Keyboard shortcuts

  • / open search mode.
  • Enter open selected bookmark, run search or copy calculated values
  • Esc clear filter or close current context.
  • ? open or close this help popup.
  • F2 toggle edit mode.
  • Click to select link or separator.
  • E to edit the selected element.
  • Arrow keys (or W, A, S, D) to move.
  • N add a new link after selected.
  • M add a separator after selected.
  • C copy/duplicate the selected element.
  • X to delete the selected.
