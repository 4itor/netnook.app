Netnook.app Help

Netnook.app is a custom new page launcher focused on privacy and speed dialing.

Netnook.app is just an open source static site, hosted on github pages.

All information is stored local-only (offline) on the user's browser. Remember to backup your bookmarks (can be exported and downloaded in a simple json format)

There is a companion chrome extension to use netnook.app as a new tabpage automatically.

Click to select link or separator
~ E to edit
~ Cursors (or wasd) to move
~ N to add link after selected
~ M add separator after
~ X to delete
